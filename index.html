Voici les règles

  • Le but du jeu est de découvrir toutes les cases libres sans cliquer sur une mine.
  • Les chiffres sur les cases indiquent le nombre de mines présentes dans les cases adjacentes.
  • Utilisez le bouton droit de la souris pour marquer une case suspectée d'être une mine.
  • Le jeu se termine si vous cliquez sur une mine.
  • Vous gagnez si vous découvrez toutes les cases sans mine.
  • Vous pouvez ajuster les dimensions de la grille et le nombre de mines avant de commencer le jeu.
  • Utilisez le bouton "Reset" pour recommencer une nouvelle partie.
